# Инструкция по созданию ИСПРАВЛЕННОГО шрифта Calibri40

## Проблема
В оригинальном шрифте Calibri отсутствуют украинские символы, из-за чего:
- "МІЗ-МА" отображается как прямоугольники (вместо украинской буквы І)
- "ТС" может отображаться неправильно

## Решение
Создан скрипт **Create_calibri40_fixed.pde** с добавленными украинскими символами.

## Настройки исправленного скрипта

- **Шрифт**: calibri.ttf
- **Размер**: 40 пикселей
- **Исправленный набор символов**: 
  - Цифры: 0-9
  - Заглавные английские: A-Z
  - Строчные английские: a-z
  - Кириллица: А-Я, а-я (включая Ё, ё)
  - **Украинские символы**: І, і (0x0406, 0x0456)
  - Все нужные знаки: пробел, !, ", $, %, ', (, ), *, +, ,, -, ., /, :, ;, =, ?, @, [, ], _, {, }, ~
- **Всего символов**: ~106 символов
- **Создание C-заголовочного файла**: включено

## Как запустить

1. **Откройте Processing IDE**
2. **Откройте скрипт** `Create_calibri40_fixed.pde`
3. **Запустите скрипт** нажатием кнопки "Run" (▶️)

## Результат

После выполнения скрипта в папке `FontFiles/` будут созданы файлы:

- `calibri40.vlw` - файл шрифта для TFT_eSPI
- `calibri40.h` - C-заголовочный файл с массивом данных шрифта

## Использование в Arduino

```cpp
#include "calibri40.h"

// Загрузка исправленного шрифта
tft.loadFont(calibri40);

// Теперь эти тексты будут отображаться правильно:
tft.print("МІЗ-МА");  // Украинская буква І будет видна
tft.print("ТС");      // Буквы Т и С будут видны
tft.print("$100 + 50% = 150");

// Освобождение памяти
tft.unloadFont();
```

## Unicode коды украинских символов

- І = 0x0406 (заглавная украинская І)
- і = 0x0456 (строчная украинская і)

## Ожидаемый размер файла

- calibri40_fixed: ~65-95 KB (чуть больше из-за украинских символов)

## Что исправлено

✅ **МІЗ-МА** - теперь украинская буква І отображается правильно
✅ **ТС** - обе буквы Т и С отображаются корректно
✅ **Все остальные символы** работают как прежде

## Замена в коде

После создания исправленного шрифта замените в main.h:

```cpp
// Было:
#define AA_FONT_CALIBRI40 calibri40

// Стало (если нужно):
#define AA_FONT_CALIBRI40 calibri40_fixed
```

Или просто замените файл `calibri40.h` на новый исправленный файл.

## Проверка

После замены шрифта проверьте отображение:
- "МІЗ-МА" должно показывать украинскую букву І
- "ТС" должно показывать обе буквы Т и С
- Все остальные символы должны работать как прежде
