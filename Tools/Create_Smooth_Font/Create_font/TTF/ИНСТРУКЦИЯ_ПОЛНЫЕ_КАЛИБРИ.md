# Инструкция по созданию ПОЛНЫХ шрифтов Calibri (12, 15, 20, 40)

## Созданные скрипты с ПОЛНЫМ набором символов

1. **Create_calibri12_full.pde** - размер 12 пикселей
2. **Create_calibri15_full.pde** - размер 15 пикселей  
3. **Create_calibri20_full.pde** - размер 20 пикселей
4. **Create_calibri40_full.pde** - размер 40 пикселей

## Настройки всех скриптов

- **Шрифт**: calibri.ttf
- **ПОЛНЫЙ набор символов**: 
  - Basic Latin (0x0021-0x007E): все печатные латинские символы
  - Cyrillic (0x0400-0x04FF): вся кириллица
- **Всего символов**: ~400+ символов
- **Создание C-заголовочного файла**: включено

## Включаемые символы

### Basic Latin (0x0021-0x007E):
- Цифры: 0-9
- Заглавные буквы: A-Z
- Строчные буквы: a-z
- Знаки препинания: ! " # $ % & ' ( ) * + , - . / : ; < = > ? @ [ \ ] ^ _ ` { | } ~

### Cyrillic (0x0400-0x04FF):
- Вся кириллица: А-Я, а-я, Ё, ё
- Дополнительные кириллические символы

## Как запустить

1. **Откройте Processing IDE**
2. **Откройте нужный скрипт** (например, Create_calibri12_full.pde)
3. **Запустите скрипт** нажатием кнопки "Run" (▶️)
4. **Повторите для каждого размера**

## Результат

После выполнения каждого скрипта в папке `FontFiles/` будут созданы файлы:

- `calibri12.vlw` и `calibri12.h`
- `calibri15.vlw` и `calibri15.h`
- `calibri20.vlw` и `calibri20.h`
- `calibri40.vlw` и `calibri40.h`

## Использование в Arduino

```cpp
#include "calibri12.h"
#include "calibri15.h"
#include "calibri20.h"
#include "calibri40.h"

// Использование разных размеров
tft.loadFont(calibri12);
tft.print("Hello World! Привет мир!");
tft.print("1234567890");
tft.print("!@#$%^&*()");

tft.loadFont(calibri15);
tft.print("УВАГА! МИЗ-МА");

tft.loadFont(calibri20);
tft.print("ТС О");

tft.loadFont(calibri40);
tft.print("Любой текст на русском и английском!");

// Освобождение памяти
tft.unloadFont();
```

## Ожидаемые размеры файлов

⚠️ **ВНИМАНИЕ**: Полные шрифты будут значительно больше!

- calibri12_full: ~50-80 KB
- calibri15_full: ~80-120 KB
- calibri20_full: ~120-180 KB
- calibri40_full: ~200-300 KB
