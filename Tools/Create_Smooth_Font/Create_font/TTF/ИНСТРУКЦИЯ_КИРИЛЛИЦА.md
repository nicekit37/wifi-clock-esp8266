# Инструкция по созданию шрифтов с кириллицей

## Проблема
Нужна кириллица для отображения "МІЗ-МА", "ТС", "УВАГА".

## Решения

### Решение 1: Минимальный шрифт с кириллицей
**Скрипт**: `Create_cyrillic_minimal.pde`
- **Шрифт**: Times New Roman (отличная поддержка кириллицы)
- **Символы**: только нужные для вашего проекта
- **Размер**: ~15-25 KB

### Решение 2: Полный шрифт с кириллицей
**Скрипт**: `Create_cyrillic_full.pde`
- **Шрифт**: Arial (хорошая поддержка кириллицы)
- **Символы**: полная кириллица + нужные знаки
- **Размер**: ~40-60 KB

## Настройки скриптов

### Минимальный набор символов:
- **Цифры**: 0-9
- **"МІЗ-МА"**: М, І, З, -, М, А
- **"ТС"**: Т, С
- **"УВАГА"**: У, В, А, Г, О
- **Знаки**: пробел, ., %, +, =, !

### Полный набор символов:
- **Цифры**: 0-9
- **Полная кириллица**: А-Я, а-я, Ё, ё
- **Украинские символы**: І, і
- **Все основные знаки**

## Как запустить

1. **Откройте Processing IDE**
2. **Выберите скрипт**:
   - `Create_cyrillic_minimal.pde` - для экономии памяти
   - `Create_cyrillic_full.pde` - для полной поддержки
3. **Запустите скрипт** нажатием кнопки "Run" (▶️)

## Проверка кириллицы

После запуска скрипта в окне предварительного просмотра должны отображаться:
- ✅ **МІЗ-МА** - все буквы видны
- ✅ **ТС** - обе буквы видны
- ✅ **УВАГА** - все буквы видны
- ✅ **Цифры и знаки** - работают корректно

## Использование в Arduino

```cpp
// Для минимального шрифта:
#include "TimesNewRoman40.h"
tft.loadFont(TimesNewRoman40);

// Для полного шрифта:
#include "Arial40.h"
tft.loadFont(Arial40);

// Тестирование кириллицы:
tft.print("МІЗ-МА");  // Должно отображаться правильно
tft.print("ТС");      // Должно отображаться правильно
tft.print("УВАГА!");  // Должно отображаться правильно

tft.unloadFont();
```

## Unicode коды кириллических символов

### Основные буквы:
- М = 0x041C, І = 0x0406, З = 0x0417, А = 0x0410
- Т = 0x0422, С = 0x0421
- У = 0x0423, В = 0x0412, Г = 0x0413, О = 0x041E

### Полная кириллица (0x0410-0x044F):
А, Б, В, Г, Д, Е, Ё, Ж, З, И, Й, К, Л, М, Н, О, П, Р, С, Т, У, Ф, Х, Ц, Ч, Ш, Щ, Ъ, Ы, Ь, Э, Ю, Я
а, б, в, г, д, е, ё, ж, з, и, й, к, л, м, н, о, п, р, с, т, у, ф, х, ц, ч, ш, щ, ъ, ы, ь, э, ю, я

## Рекомендации

1. **Сначала попробуйте минимальный шрифт** - экономит память
2. **Если нужна полная поддержка** - используйте полный шрифт
3. **Проверьте предварительный просмотр** - символы должны отображаться правильно
4. **Times New Roman** имеет отличную поддержку кириллицы
5. **Arial** также хорошо поддерживает кириллицу

## Ожидаемые размеры файлов

- **Минимальный**: ~15-25 KB
- **Полный**: ~40-60 KB

## Замена в коде

После создания шрифта замените в main.h:

```cpp
// Было:
#define AA_FONT_CALIBRI40 calibri40

// Стало (для минимального):
#define AA_FONT_CALIBRI40 TimesNewRoman40

// Или (для полного):
#define AA_FONT_CALIBRI40 Arial40
```

Теперь кириллица должна отображаться правильно! 🎉
