# Руководство по созданию правильного digital75 шрифта

## 🎯 Цель
Создать правильный TFT_eSPI шрифт digital75 из TTF файла digital_7.ttf для использования с RLE-прокси.

## 📋 Пошаговая инструкция

### 1. Откройте Processing IDE
- Запустите Processing IDE
- Откройте файл `Create_font.pde`

### 2. Проверьте настройки
Убедитесь, что в файле установлены следующие параметры:

```processing
// Настройки шрифта
String fontName = "digital_7";  // Используем digital_7.ttf из папки data
String fontType = ".ttf";
int  fontSize = 75;  // Размер шрифта 75 пикселей

// Unicode блоки - только цифры 0-9
static final int[] unicodeBlocks = {
  0x0030, 0x0039, //Только цифры 0-9 для экономии памяти
};

// Дополнительные символы для температуры
static final int[] specificUnicodes = {
  0x002E, 0x002D, // . - (точка и минус для температуры)
};
```

### 3. Запустите скетч
- Нажмите **Ctrl+R** или кнопку "Run"
- Дождитесь завершения обработки
- Проверьте консоль на наличие ошибок

### 4. Проверьте созданные файлы
После успешного выполнения в папке `FontFiles` должны появиться:
- `digital75.h` - заголовочный файл для TFT_eSPI
- `digital75.vlw` - файл шрифта Processing

### 5. Скопируйте файл в проект TS
```bash
copy FontFiles\digital75.h ..\..\TS\src\digital75_new.h
```

## 🔍 Проверка результата

### Структура правильного TFT_eSPI шрифта:
```cpp
const uint8_t digital75[] PROGMEM = {
  // Заголовок (16 байт):
  // [0-3]   - размер данных шрифта (big endian)
  // [4-7]   - количество символов (big endian)  
  // [8-11]  - высота шрифта (big endian)
  // [12-15] - флаги (big endian)
  
  // Таблица символов (16 байт на символ):
  // [0-3]   - код символа (big endian)
  // [4-7]   - ширина символа (big endian)
  // [8-11]  - высота символа (big endian) 
  // [12-15] - смещение данных символа (big endian)
  
  // Данные символов (пиксели)
};
```

### Ожидаемые характеристики:
- **Количество символов**: 12 (цифры 0-9 + точка + минус)
- **Высота шрифта**: 75 пикселей
- **Размер файла**: ~50-100 КБ (зависит от сложности символов)

## ❗ Возможные проблемы

### Ошибка "Font not found"
- Убедитесь, что файл `digital_7.ttf` находится в папке `data`
- Проверьте правильность имени файла (без расширения в fontName)

### Ошибка "Out of memory"
- Уменьшите размер шрифта (fontSize)
- Исключите ненужные Unicode блоки

### Неправильная структура файла
- Убедитесь, что используется правильный TTF файл
- Проверьте настройки Unicode блоков
